The RestroPress - Menu Cart plugin is useful for the RestroPress users who want to show a cart menu on the header.

== Description ==

Required Plugin:
- You need to active Restropress plugin to use this plugin

Checkout -> <a href = "https://wordpress.org/plugins/restropress/" target="_blank"> Download RestroPress</a>

Available features:
- You can chnage the cart icon image by choose available icon option.
- Setting option provided to show cart quantity with cart total price.
- Manage to show the cart icon as per the cart value is empty or not.

Note:- To view the cart menu, you must have a Primery menu. If you have not created any primery menu then Go through this setting path Appearance -> Menu -> Here you need to create a new menu and set that as Primery menu.

= What is RestroPress and how to download RestroPress? =

RestroPress is an online food ordering system for WordPress. It is a standalone WordPress plugin that allows you to easily add a food ordering system to your WordPress website. Using RestroPress, you can easily receive both Pickup/Takeaway and Delivery orders.

RestroPress has a user-friendly frontend and backend interface that will allow you to easily manage orders and also comes with a pre-built user dashboard to manage their profile and orders.

The plugin automatically adds the required pages to your site when you activate it. The plugin displays the food items on your page using the [fooditems] shortcode.

[Download RestroPress](https://restropress.com/checkout?edd_action=add_to_cart&download_id=53444)

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "RestroPress - Menu Cart".
3. Click install.
4. Click activate.
5. Once the plugin is installed then you can see RestroPress on the left navigation bar of WordPress Dashboard.
